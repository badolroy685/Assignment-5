Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM, such as calling ByID, By Class Name, By Tag name, By Name Attribute, using Query Selector.


Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets the entire HTML structure, while innerText only sets the visible text of an element. On the other hand textContent sets the text content of an element, regardless of style or visibility.


Question-3: What is event delegation in the DOM?

Event delegation is a useful event handling pattern in the DOM that used to attach a single event listener to a parent, and using it to manage events for its current or future child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Add elements can be create by using appendChild(). Example: parentElement.appendChild(newElement);
Remove element can be create by using removeChild(). Example: parentElement.removeChild(childElement);
